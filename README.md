
# AKAN NAME DETERMINER WEB APPLICATION


This web application is based on determining the Akan name of a user with regards to his or her inputs.
The user is therefore required to key in their Gender and their birthday from the form provided. This web application then calculates the day of the week of the user was born, then depending on their gender outputs it provides their Akan name.
